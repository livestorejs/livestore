# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities privately via
[GitHub Security Advisories](https://github.com/livestorejs/livestore/security/advisories/new)
for this repository. Please do not open public issues for security problems.

Reports are handled by the project maintainer on a best-effort basis; you
should normally hear back within a few days.

## Supported versions

Security fixes are applied to the latest release line only.
