# Diagrams

- We're using [D2](https://d2lang.com/) to create some of the diagrams in the docs.
  - Make sure to study the [D2 documentation](https://d2lang.com/tour/sequence-diagrams/) to understand the syntax - see [page sitemap](https://d2lang.com/sitemap.xml) for all pages.
  - Make sure to always include the `base.d2` file in the diagram via the `...@../../../base.d2` include directive.

- We're still using Mermaid for some other diagrams.
- We're also using TLDraw for some other diagrams.
